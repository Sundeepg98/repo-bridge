# repo-bridge

Landing page for the **`sundeepg98-repo-bridge`** GitHub App — a personal app that bridges
private repositories into AI sandboxes (e.g. claude.ai chat tabs) using GitHub's
device-flow authorization: scoped, short-lived, per-repo access with no long-lived token.

Served via GitHub Pages → https://sundeepg98.github.io/repo-bridge/

This repo intentionally contains only the static landing page. The app itself is managed in
GitHub settings; the repositories it can reach stay private.
