GitHub App · @Sundeepg98

sundeepg98-repo-bridge

A private, two-way span between the repositories you own and the AI sandboxes that work on them — clone down, push back.

  • Device flow

    You approve one sign-in at github.com/login/device. No password or token is ever pasted into the sandbox.

  • Two-way & scoped

    A short-lived token grants read and write to only the repositories you select — clone down and push back — then it expires on its own.

  • Stays private

    Your repositories remain private. Access is granted per repo and is revocable anytime from GitHub settings.
