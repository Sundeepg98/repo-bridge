GitHub App · @Sundeepg98

sundeepg98-repo-bridge

A private span between the repositories you own and the AI sandboxes that work on them.

  • Device flow

    You approve one sign-in at github.com/login/device. No password or token is ever pasted into the sandbox.

  • Scoped & brief

    A short-lived token is minted for only the repositories you select, then expires on its own — never an all-access personal token.

  • Stays private

    Your repositories remain private. Access is granted per repo and is revocable anytime from GitHub settings.
